Hotel Aqua

El objetivo de este proyecto era crear una página web moderna y atractiva para el Hotel Aqua, que pudiera ofrecer una experiencia de usuario fluida y efectiva para sus clientes potenciales. Como programador, mi papel en este proyecto fue desarrollar y diseñar la página web desde cero.

View Technology

La página web cuenta con un diseño atractivo y moderno. Así como un menú de navegación intuitivo que permite a los usuarios acceder fácilmente a la información que necesitan. La página web se desarrolló utilizando tecnologías de desarrollo web, como HTML5, CSS3 y JavaScript.

Sivana

La página web se desarrolló utilizando tecnologías de desarrollo web de última generación, como HTML5, CSS3 y JavaScript. También se utilizó una estructura de diseño responsivo para garantizar que la página web sea accesible y funcione correctamente en dispositivos móviles y tablets.
